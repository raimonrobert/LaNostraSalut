# Changelog — LaNostraSalut

Formato: `[versión] — fecha — descripción`

---

## [1.0.0] — juny 2026 — Versió inicial

### Funcionalitats
- Agenda de cites mèdiques amb 4 pacients i 5 acompanyants
- Filtres per persona i per cites passades
- Vista de llista i vista de calendari mensual
- Assignació d'acompanyant per cita ("M'apunto")
- Notes pre i post visita
- Sincronització en temps real via Supabase
- Gestió de pacients i acompanyants des dels ajustos
- Pantalla d'inici "Qui ets?" amb selecció per nom
- Identificació visual per cercles de color amb inicial
- Disseny mobile-first, instal·lable com a PWA

### Stack
- HTML + JS vanilla (fitxer únic)
- Supabase free tier (PostgreSQL + Realtime)
- GitHub Pages per al hosting
